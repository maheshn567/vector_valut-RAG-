API Documentation Quickstart

Use these endpoints to interact with your Vector Vault corpora. Base URL: https://api.vectorvault.io

POST /api/v1/rag/create

Ingest Document

cURL 
curl -X POST \
  https://api.vectorvault.io/api/v1/rag/create \
  -H 'Authorization: Bearer YOUR_API_KEY' \
  -H 'Content-Type: application/json' \
  -d '{
    "corpus_id": "corp_9x2...",
    "content": "Vector databases...",
    "metadata": {"source": "wiki"}
  }'
POST /api/v1/query/ask

Generate RAG Answer

import requests

url = "https://api.vectorvault.io/api/v1/query/ask"
headers = {
    "Authorization": "Bearer YOUR_API_KEY",
    "Content-Type": "application/json"
}
data = {
    "corpus_id": "corp_9x2b4",
    "query": "Explain the difference between HNSW and IVFFlat?",
    "temperature": 0.2
}

response = requests.post(url, headers=headers, json=data)
print(response.json())
